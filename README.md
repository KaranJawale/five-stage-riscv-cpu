# five-stage-riscv-cpu

A 5-stage pipelined RISC-V processor implemented in Verilog.

## Features

- IF, ID, EX, MEM, WB stages
- Separate instruction and data memories
- Support for:
  - ADD
  - SUB
  - AND
  - OR
  - LD
  - SD

## Future Improvements

- Data forwarding
- Hazard detection
- Branch handling
- Branch prediction
